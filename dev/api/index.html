YUVColorTypes — Module
YUVColorTypes

An add-on libray to ColorTypes supporting YUV-like colors.

Basic Types

YUVColorTypes.YUV — Type
YUV{T,P,U,D} <: AbsrtractYUV{T}
  • T: component type
  • P: symbol of profile
  • U: unsigned chroma components if true
  • D: bit-depth

See also AYUV and YUVA for transparent versions, and Luma for the single Y component version.

source
YUVColorTypes.YCbCr30BT601_625 — Type
YCbCr30BT601_625{T}

10-bit representation of the 625-line BT.601 YCbCr color type. Its raw components of type T usually have a value in [0, 1024).

source
YUVColorTypes.YCbCr24BT601_625 — Type
YCbCr24BT601_625{T}

8-bit representation of the 625-line BT.601 YCbCr color type. Its raw components of type T usually have a value in [0, 256). See also YCbCr30BT601_625.

source
